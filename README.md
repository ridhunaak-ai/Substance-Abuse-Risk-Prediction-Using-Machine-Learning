# Substance Abuse Risk Prediction Using Machine Learning

## Overview

Substance Abuse Risk Prediction Using Machine Learning is a predictive analytics project designed to assess the likelihood of substance abuse based on behavioral, demographic, academic, and lifestyle factors. The system leverages machine learning techniques to identify risk patterns and generate accurate predictions that can support early intervention and informed decision-making.

## Features

* Data preprocessing and cleaning
* Exploratory Data Analysis (EDA)
* Feature engineering and transformation
* Machine learning model training and evaluation
* Risk prediction based on user inputs
* Interactive web application using Streamlit
* Real-time prediction results

## Dataset

The project utilizes a structured dataset containing demographic, behavioral, and lifestyle attributes associated with substance abuse risk. Data preprocessing techniques were applied to handle missing values, encode categorical variables, and scale numerical features.

## Installation

Clone the repository:

```bash
git clone https://github.com/ridhunaak-ai/Substance-Abuse-Risk-Prediction-Using-Machine-Learning.git
cd Substance-Abuse-Risk-Prediction-Using-Machine-Learning
```

Install dependencies:

```bash
pip install -r requirements.txt
```

## Run the Application

```bash
streamlit run app4.py
```

The application will launch in your browser and allow users to enter relevant information for substance abuse risk prediction.
